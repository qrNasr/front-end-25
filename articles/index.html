1. HTML: The Structure

HTML uses "tags" (like <p>, <h1>, <div>) to define different parts of your content. Tags usually come in pairs: an opening tag (<tagname>) and a closing tag (</tagname>).

Look at the code of this page (the file you saved). You'll see tags like:

  • <!DOCTYPE html>: Declares it's an HTML5 document.
  • <html>: The root element of the page.
  • <head>: Contains meta-information (like the title, links to CSS/JS) - not directly visible.
  • <body>: Contains the visible content of the page (text, images, links, etc.).
  • <h1> to <h6>: Headings (<h1> is the most important).
  • <p>: Paragraphs of text.
  • <ul> (Unordered List) & <li> (List Item): For bullet points.
  • <a href="...">: Creates hyperlinks (links).
  • <div>: A generic container used to group elements (often for styling).

Example HTML Snippet:

<h2>This is a Heading</h2>
<p>This is a paragraph with a <a href="#">link</a>.</p>
<ul>
  <li>First item</li>
  <li>Second item</li>
</ul>

HTML provides the essential foundation for your webpage content.

2. CSS: The Style

CSS makes your HTML look good! It controls colors, fonts, spacing, layout, and more. Without CSS, web pages would be plain black text on a white background.

CSS works by selecting HTML elements and applying "rules" (properties and values) to them.

  • Selectors: Target specific HTML elements (e.g., all <p> tags, elements with a specific class like .container, or an element with a unique ID like #intro).
  • Properties: What you want to change (e.g., color, font-size, background-color, margin).
  • Values: The specific setting for the property (e.g., blue, 16px, #f0f0f0, 10px).

In this file, the CSS is located inside the <style> tags within the <head> section. Look up and see how different elements (like body, h1, .container) are being styled.

Example CSS Rule:

/* Select all paragraph tags */
p {
  color: navy; /* Set the text color to navy */
  font-size: 16px; /* Set the font size */
}

/* Select elements with the class 'highlight' */
.highlight {
  background-color: yellow;
}

CSS brings visual design and layout to your structured HTML content.

3. JavaScript: The Interactivity

JavaScript (JS) adds behavior and interactivity to your website. It can:

  • Respond to user actions (like clicks, mouse movements, keyboard presses).
  • Change HTML content and CSS styles dynamically (without reloading the page).
  • Perform calculations, validate forms, fetch data, create animations, and much more!

JavaScript code is usually placed within <script> tags, often just before the closing </body> tag, or linked from an external .js file.
