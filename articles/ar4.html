    Semantic HTML5 Tags

    These tags provide meaning about the structure of the content, helping browsers, search engines, and assistive technologies understand your page better.

    • : Represents introductory content or navigational aids (often contains H1, logo, nav).
    • : Contains major navigation links.
    • : Specifies the main, dominant content of the document.
    • : Represents a self-contained composition (e.g., blog post, forum post, news story).
    • : Represents a standalone thematic section within a document.
    • : Represents content tangentially related to the main content (e.g., sidebar, pull quotes).
    • : Represents the footer for its nearest sectioning content or the whole page (often contains copyright, related links).
